# 🛒 ShopSphere – AI Powered Full Stack MERN E-Commerce Platform

## 🚀 Overview

ShopSphere is a full-stack MERN e-commerce platform that allows users to browse, search, purchase, and manage products across multiple categories. The platform includes secure authentication, Razorpay payment integration, order management, admin dashboard, analytics, and an AI-powered shopping assistant chatbot using Google Gemini.

## 🛠️ Tech Stack

**Frontend:** React.js, Redux Toolkit, React Router, Axios, Tailwind CSS
**Backend:** Node.js, Express.js, JWT, bcrypt
**Database:** MongoDB, Mongoose
**Cloud Storage:** Cloudinary
**Payments:** Razorpay
**AI Integration:** Google Gemini API
**Deployment:** Vercel, Render, MongoDB Atlas
